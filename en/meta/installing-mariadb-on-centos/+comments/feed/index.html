MariaDB Knowledge Base Comments for: Installing MariaDB on CentOShttps://mariadb.com/kb/en/installing-mariadb-on-centos/+comments/feed/en-usWed, 15 Mar 2023 02:33:31 +0000Re: Installing MariaDB on CentOShttps://mariadb.com/kb/en/installing-mariadb-on-centos/+comments/929<p>Hi, </p> <p>As far as I can see, CentOS comes with MySQL 5.1, so you would most likely have a similar problem upgrading to MySQL 5.5 as well.</p> <p>You need to remove older MySQL packages first, e.g. to run </p> <pre> sudo yum remove mysql-server mysql </pre> <p>(and whichever else mysql* packages cause the conflict), and then install MariaDB server as you already did. </p> <p>Don't forget to backup your data before the upgrade, it is always the good practice. </p> Elena Stepanovahttps://mariadb.com/kb/en/installing-mariadb-on-centos/+comments/929
