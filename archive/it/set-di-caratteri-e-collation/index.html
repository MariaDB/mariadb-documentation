Set di Caratteri e Collation

In parole povere, un set di caratteri stabilisce quali caratteri e come vengono resistrati per supportare una o più lingue. Una collarion invece determina l'ordine da usare nel comparare le stringhe (ovvero l'ordine di un determinato carattere nell'alfabeto usato da quella lingua).
