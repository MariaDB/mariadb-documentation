WiredTree aggiunge MariaDB per aumentare le performance dei database

WiredTree, un esperto provider di server virtuali privati (VPS hosting), server dedicati e server ibridi, ha annunciato la disponibilità di MariaDB su tutti i piani di hosting senza costi aggiuntivi per i clienti.

Tra le motivazioni legate alle performance che sono citate nella conferenza stampa:

  • "MariaDB è un'alternativa a MySQL veloce e facile da implementare, che noi crediamo possa massimizzare le performance dei siti web," afferma Zac Cogswell, presidente di WiredTree. "Dalla gestione dei contenuti alle message board come Joomla o vBulletin, i nostri clienti di hosting sui server dedicati necessitano di uno storage di database robusti che possano far fronte alla domanda. MariaDB offre le stesse funzionalità di MySQL, ma con degli storage engine ottimizzati per rendere le query più veloci che mai."
  • "Prendiamo vBulletin per esempio," afferma Cogswell. "Le sessioni aperte sono mantenute nelle tabelle heap, che non hanno indici clusterizzati. Di conseguenza l'elaborazione è lenta. Le tabelle heap sono un punto debole comune per i database, ma MariaDB le velocizza significativamene. Un'istruzione INSERT può essere più veloce del 24%, mentre gli indici possono essere più rapidi del 60%. I nostri clienti che utilizzano vBulletin, Joomla, o qualsiasi applicazione che faccia uso delle tabelle heap, noterà un notevole miglioramento delle prestazioni. Come risultato, MariaDB può mantenere la sua velocità, anche con molte connessioni concorrenti."

Si veda il comunicato stampa per ulteriori informazioni.
