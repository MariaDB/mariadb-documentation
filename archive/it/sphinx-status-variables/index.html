Le variabili di stato di Sphinx

Contents

  1. Sphinx_error
  2. Sphinx_time
  3. Sphinx_total

  5. Sphinx_word_count

Questa pagina documenta le variabili di stato dello storage engine Sphinx. si veda la pagina Server Status Variables per un elenco completo delle variabili di stato che possono essere visualizzate con SHOW STATUS.

Si veda anche Full list of MariaDB options, system and status variables.

Sphinx_error

  • Spiegazione: Si veda SHOW ENGINE SPHINX STATUS.
  • Scopo: Globale, Sessione
  • Tipo: numerico
  • Introdotta in: MariaDB 5.5

Sphinx_time

  • Spiegazione: Si veda SHOW ENGINE SPHINX STATUS.
  • Scopo: Globale, Sessione
  • Tipo: numerico
  • Introdotta in: MariaDB 5.5

Sphinx_total

  • Spiegazione: Si veda SHOW ENGINE SPHINX STATUS.
  • Scopo: Globale, Sessione
  • Tipo: numerico
  • Introdotta in: MariaDB 5.5

Sphinx_total_found

  • Spiegazione: Si veda SHOW ENGINE SPHINX STATUS.
  • Scopo: Globale, Sessione
  • Tipo: numerico
  • Introdotta in: MariaDB 5.5

Sphinx_word_count

  • Spiegazione: Si veda SHOW ENGINE SPHINX STATUS.
  • Scopo: Globale, Sessione
  • Tipo: numerico
