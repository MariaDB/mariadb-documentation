Contare quanti record sono stati cancellati

Buongiorno a tutti,

premessa: non dispongo di skill su db in generale

database: mysql Ver 15.1 Distrib 5.5.60-MariaDB, for Linux (x86_64) using readline 5.1

requisito: non posso adottare script php o altro, devo risolvere con le query

obiettivo: conoscere il numero di rek che vengono cancellati da una query

spiegazione: ho un db che deve essere svecchiato dei rek più vecchi di 180 giorni ho disposto la seguente query che soddisfa le mie richieste: DELETE FROM SystemEvents WHERE (DeviceReportedTime<(now()-INTERVAL 180 DAY)); ora desidero conoscere quanti rek vengono cancellati dalla query e mi domando se questo è possibile senza lanciare 2 query come ho fatto fino ad ora (prima la SELECT poi la DELETE)

Ovviamente ho cercato in rete ma non ho trovato nulla che mi possa aiutare e per questo motivo mi rivolgo qui Ringrazio per la cortese collaborazione Dario
