HeidiSQL

HeidiSQL è un client Windows per MariaDB e MySQL. E' stato inserito nella versione Windows di MariaDB a partire da MariaDB 5.2.7.

Versione di MariaDBVersione di HeidiSQL Inclusa
MariaDB 5.5.31HeidiSQL 8.0

HeidiSQL è in grado di:

  • Connettersi a più server in un'unica finestra
  • Connettersi ai server attraverso la riga di comando
  • Creare e modificare tabelle, viste, stored routine, trigger ed eventi.
  • Esportare in SQL
  • Esportare da un server/database direttamente in un altro server/database
  • Gestire i privilegi utente
  • Importare file di testo
  • Esportare le righe come CSV, HTML, XML, SQL, LaTeX e Wiki Markup
  • Visualizzare e modificare i dati delle tabelle usando comode griglie
  • Modifiche di massa (spostare in un altro db, cambiare engine, collation, etc.)
  • Inserimenti batch di file ascii o binari nelle tabelle
  • Scrivere query con autocompletamento e sintassi evidenziata personalizzabili
  • Riformattazione dell'SQL disordinato
  • Monitorare e terminare i processi client
  • Trovare un testo specifico in tutte le tabelle di tutti i database di un server
  • Ottimizzare e riparare tabelle in batch
  • Altro...

Ulteriori informazioni, compresi alcuni Screenshot di HeidiSQL sono disponibili sul sito di HeidiSQL.
