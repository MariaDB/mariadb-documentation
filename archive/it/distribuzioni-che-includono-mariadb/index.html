Distribuzioni che includono MariaDB

Le seguenti distribuzioni includono MariaDB nei loro repository. Ciò permette di usare il gestore dei pacchetti di sistema.

Distribuzioni Linux

  • GNU/Linux KDu — contiene l'ultimo rilascio di MariaDB 5.5, che sostituisce MySQL per default a partire dalla versioneGNU/Linux KDu 2.0 Final (non ancora uscita)
  • Mageia — include la versione più recente di MariaDB 5.5, che sostituisce MySQL per default
  • Gentoo
  • openSUSE — MariaDB 5.5 ora è il database "MySQL" di default in OpenSuse 12.3. MariaDB 10.0 è presente nel repository obs
  • Slackware
  • ArchLinux — presente nel repo community
  • ALTLinux
  • Immagini Live di Debian con MariaDB
  • Parabola GNU/Linux
  • TurboLinux
  • The Chakra Project — contiene l'ultima release di MariaDB 5.5, che sostituisce MySQL per default
  • Fedora — MariaDB 5.5 è ora il "database MySQL" di default, a partire da Fedora 19 ¶
  • Red Hat Enterprise Linux — MariaDB 5.5 è ora il "database MySQL" di default, a partire da RHEL 7

Distribuzioni BSD

  • FreeBSD — MariaDB 5.3 è presente nell'albero dei port, e il manuale di FreeBSD contiene il capitolo: Installing Applications: Packages and Ports
    • MariaDB su FreshPorts
  • OpenBSD — MariaDB 5.5 sostituirà il port attuale di MySQL 5.1 nella prossima release.

Distribuzioni Mac OS X

  • Homebrew. Un buon blog è reperibile qui. Se Homebrew è installato, è possibile installare MariaDB così: brew install mariadb. Ulteriori informazioni: Fare il build di MariaDB su MacOS X utilizzando Homebrew.
  • MacPorts. This provides mariadb and mariadb-server. A quick guide on how to install it.
