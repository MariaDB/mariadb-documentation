mariasql per node.js

mariasql è un binding di node.js alla libreria client non-bloccante di MariaDB. E' compatibile con MySQL.

E' diverso dagli altri binding di libmysqlclient perché utilizza le funzioni non-bloccanti della libreria client di MariaDB.

E' possibile utilizzarla con il npm package installer: npm install mariasql

Nei benchmark, le prestazioni di mariasql superano di molto quelle di libmysqlclient.

Il codice sorgente si trova su github:node-mariasql.
