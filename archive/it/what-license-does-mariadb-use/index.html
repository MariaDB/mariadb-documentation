Quale licenza usa MariaDB?

MariaDB è disponibile secondo i termini della licenza GPL v2, come MySQL.

Alcune porzioni di codice utilizzano la licenza BSD o sono di pubblico dominio. Si vedano i file degli header per i dettagli.

Stiamo lavorando per creare una libreria C LGPL che sarà compatibile con quella attuale. Nel frattempo è possibile trovare una vecchia libreria client LGPL qui.

Chi desidera essere parte della comunità e donare del codice al progetto MariaDB, deve rilasciare il codice al progetto con licenza BSD-NEW o firmare il Monty Program Contributor Agreement e spedirlo via email. Il motivo è che gli sviluppatori desiderano poter fornire il codice ad altri che non possono usarlo come GPL, ad esempio il progetto MySQL di Oracle.
