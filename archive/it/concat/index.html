  • CONCAT [en]

CONCAT

Syntassi

CONCAT(str1,str2,...)

Spiegazione

Restituisce la stringa risultante dalla concatenazione degli argomenti. Accetta uno o più argomenti. Se tutti gli argomenti sono stringhe non binarie, il risultato è una stringa non binaria. Se gli argomenti comprendono anche stringhe binarie, il risultato è una stringa binaria. Un argomento numerico è convertito nell'equivalente binario; se si desidera evitarlo, si può utilizzare una conversione esplicita, come nel seguente esempio:

SELECT CONCAT(CAST(col_int AS CHAR), col_char);

CONCAT() restituisce NULL se un qualsiasi argomento è NULL.

Esempi

MariaDB [(none)]> SELECT CONCAT('Ma', 'ria', 'DB');
+---------------------------+
| CONCAT('Ma', 'ria', 'DB') |
+---------------------------+
| MariaDB                   |
+---------------------------+
1 row in set (0.00 sec)

MariaDB [(none)]> SELECT CONCAT('Ma', 'ria', NULL, 'DB');
+---------------------------------+
| CONCAT('Ma', 'ria', NULL, 'DB') |
+---------------------------------+
| NULL                            |
+---------------------------------+
1 row in set (0.00 sec)

MariaDB [(none)]> SELECT CONCAT(42.0);
+--------------+
| CONCAT(42.0) |
+--------------+
| 42.0         |
+--------------+
1 row in set (0.00 sec)
