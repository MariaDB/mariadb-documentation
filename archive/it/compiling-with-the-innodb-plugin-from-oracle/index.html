Compilare con il plugin InnoDB di Oracle

Normalmente MariaDB è distribuito con XtraDB, una versione migliorata del plugin InnoDB di Oracle.

Se si desidera usare InnoDB di Oracle, occorre compilare MariaDB senza specificare la seguente opzione in fase di configurazione: --without-plugin-innodb_plugin. Per esempio, il semplice comando ./configure senza alcuna opzione funzionerà.

Se si compila InnoDB, il test innodb_plugin controllerà che InnoDB sia presente in aggiunta a XtraDB:

./mysql-test-run --suite=innodb_plugin

Per utilizzare innodb_plugin invece di xtradb (da MariaDB 5.5) è possibile eseguire:

mysqld --ignore-builtin-innodb --plugin-load=innodb=ha_innodb.so \
--plugin_dir=/usr/local/mysql/lib/mysql/plugin

Per MariaDB 5.3 e precedenti, il nome della libreria è ha_innodb_plugin.so

Vedi anche

http://dev.mysql.com/doc/refman/5.1/en/replacing-builtin-innodb.html
