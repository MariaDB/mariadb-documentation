Il comando HELP

Sintassi

MariaDB [(none)]> HELP termine_di_ricerca

Descrizione

Il comando HELP può essere utilizzato con qualsiasi client di MariaDB / MySQL, come il client da riga di comando mysql, per ottenere un help di base sulla sintassi SQL e una breve descrizione della maggior parte dei comandi e delle funzioni.

Se si specifica un argomento al comando help, il client mysql lo usa come termine di ricerca per accedere all'help lato server. Questo comando richiede che le tabelle di help nel database mysql siano popolate.

Se un termine non viene trovato, la ricerca fallisce:

MariaDB [(none)]> HELP qualcosa

Nothing found
Please try to run 'help contents' for a list of all accessible topics

Si può usare help contents per visualizzare un elenco degli argomenti:

MariaDB [(none)]> HELP contents
You asked for help about help category: "Contents"
For more information, type 'help <item>', where <item> is one of the following
categories:

   Account Management
   Administration
   Compound Statements
   Data Definition
   Data Manipulation
   Data Types
   Functions
   Functions and Modifiers for Use with GROUP BY
   Geographic Features
   Language Structure
   Plugins
   Table Maintenance
   Transactions
   User-Defined Functions
   Utility

Se la stringa di ricerca corrisponde a più di un termine, mysql li mostra tutti:

MariaDB [(none)]> help create
Many help items for your request exist.
To make a more specific request, please type 'help <item>',
where <item> is one of the following
topics:
   CREATE DATABASE
   CREATE EVENT
   CREATE FUNCTION
   CREATE FUNCTION UDF
   CREATE INDEX
   CREATE PROCEDURE
   CREATE SERVER
   CREATE TABLE
   CREATE TABLESPACE
   CREATE TRIGGER
   CREATE USER
   CREATE VIEW
   SHOW
   SHOW CREATE DATABASE
   SHOW CREATE EVENT
   SHOW CREATE FUNCTION
   SHOW CREATE PROCEDURE
   SHOW CREATE TABLE
   SPATIAL

Si può usare un argomento come stringa di ricerca per vedere gli elementi che contiene.

L'help è distribuito con il server MariaDB e le tabelle di help possono essere popolate con gli script mysql_install_db o fill_help_table.sql.
