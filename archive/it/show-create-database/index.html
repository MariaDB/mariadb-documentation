SHOW CREATE DATABASE

Sintassi

SHOW CREATE {DATABASE | SCHEMA} nome_db

Spiegazione

Mostra l'istruzione CREATE DATABASE che crea il database specificato. SHOW CREATE SCHEMA è sinonimo di SHOW CREATE DATABASE. SHOW CREATE DATABASE decide se virgolettare il nome del database in base alla variabile sql_quote_show_create.

Esempi

SHOW CREATE DATABASE test;
+----------+-----------------------------------------------------------------+
| Database | Create Database                                                 |
+----------+-----------------------------------------------------------------+
| test     | CREATE DATABASE `test` /*!40100 DEFAULT CHARACTER SET latin1 */ |
+----------+-----------------------------------------------------------------+

SHOW CREATE SCHEMA test;
+----------+-----------------------------------------------------------------+
| Database | Create Database                                                 |
+----------+-----------------------------------------------------------------+
| test     | CREATE DATABASE `test` /*!40100 DEFAULT CHARACTER SET latin1 */ |
+----------+-----------------------------------------------------------------+

Con sql_quote_show_create impostata a OFF:

SHOW CREATE DATABASE test;
+----------+---------------------------------------------------------------+
| Database | Create Database                                               |
+----------+---------------------------------------------------------------+
| test     | CREATE DATABASE test /*!40100 DEFAULT CHARACTER SET latin1 */ |
+----------+---------------------------------------------------------------+
