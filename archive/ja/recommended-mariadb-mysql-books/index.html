MariaDB / MySQL 推薦書籍

目次

  1. MariaDB / MySQL のエンドユーザー向け
  2. MariaDB や MySQL 上でコーディングしたい開発者向け

下記は MariaDB と MySQL に関する推薦書籍です。Amazon.co.jp へリンクを張ってありますが他に多くの書店 (ネット、実店舗) で購入出来ます。

他にも掲載して欲しい MySQL / MariaDB 関連書籍がある場合はコメントを残してください。

MariaDB / MySQL のエンドユーザー向け

MariaDB Essentials (著) Emilien Kenler, Federico Razzoli, 2015年10月
SQL を知らないけれどすぐに MariaDB をやってみたい、学習したい人へ。MySQL を知っているけれど、もっと深く入っていきたい方へ。

Getting Started with MariaDB — Second Edition (著) Daniel Bartholomew, 2015年6月
MariaDB のビギナーズガイド。MariaDB や MySQL の経験がなくても問題ありません。初版からの増補版です。

Learning MySQL and MariaDB (著) Russell J. T. Dyer, 2015年4月
MySQL と MariaDB の入門者向け。データベースシステム、データベース言語 SQL、インストールと基本的な管理について。

MariaDB High Performance (著) Pierre Mavro, 2014年9月
MariaDB の知識をもつシステム管理者・設計者、データベース管理者のために。通信量の増加に対応するため、現在のインフラをどのように拡大すればよいか。

Mastering MariaDB (著) Federico Razzoli, 2014年9月
MariaDB サーバやサーバ群の管理方法について学びたい中級者向け。

MariaDB Beginners Guide (著) Rodrigo Ribeiro Gonçalves, 2014年8月
MariaDB と SQL の初心者向けの本。

MariaDB Cookbook (著) Daniel Bartholomew, 2014年3月
MariaDB の便利なレシピが詰め込まれたクックブック。Sphinx、CONNECT、Cassandra、仮想列・動的列、TokuDB などについて書かれた章があります。

Getting Started with MariaDB (著) Daniel Bartholomew, 2013年10月
MariaDB のビギナーズガイド。MariaDB や MySQL の経験がなくても問題ありません。

Building a Web Application with PHP and MariaDB: A Reference Guide (著) Sai Srinivas Sriparasa, 2014年6月
PHP と MariaDB によるスケーラブルでセキュアな Web アプリケーションを作るための入門書。

MariaDB Crash Course (著) Ben Forta, 2011年9月
最初の MariaDB 本。
SQL と MariaDB の基本を学習したい人向け。

SQL-99 Complete, Really (著) Peter Gulutzan, Trudy Pelzer, 1999年3月
SQL:1999 (SQL99標準) のすべてを網羅した優れた書籍です。
当ナレッジベースで無償公開されています。

MySQL (5th Edition) (著) Paul DuBois, 2013年4月
The 'default' book to read if you want to learn to use MySQL / MariaDB.
MySQL・MariaDB の使い方を学びたいときに、デフォルトで読むべき本です。

MySQL Cookbook (著) Paul DuBois, 2014年8月
MySQL の使用例がたくさん載っています。Paul 氏の他の本と同様、無味乾燥なタイトルは重要であり、楽しく読むことができます。

実践ハイパフォーマンスMySQL 第3版 (著) Baron Schwartz, Peter Zaitsev, Vadim Tkachenko, (訳) 菊池 研自, 株式会社クイープ, 2013年11月
実践ハイパフォーマンスMySQL 第2版 (著) Baron Schwartz, Peter Zaitsev, Vadim Tkachenko, Jeremy D. Zawodny, Arjen Lentz, Derek J. Balling, 他, (訳) 伊藤 直也, 田中 慎司, 吉川 英興, 株式会社クイープ, 2009年12月
「High Performance MySQL は、MySQL による高速で信頼性の高いシステムを構築する決定的なガイドです。実社会で巨大なシステムを長年構築してきた著名な専門家によって執筆されたこの本は、MySQL のパフォーマンスに関するあらゆる側面から詳細にカバーし、堅牢性、安全性、データ整合性に焦点を当てています。MySQL のフルパワーを引き出すことができるように、高度なテクニックを深く学びましょう。」(O'Reilly の本の説明より)

MySQL Admin Cookbook (著) Daniel Schneller, 2010年3月
MySQL ユーザーとデータベース管理者向けの段階的クイックガイド。MySQL の設定や管理に関する実社会での課題に取り組む人に。

MariaDB や MySQL 上でコーディングしたい開発者向け

Understanding MySQL Internals (著) Sasha Pachev, former MySQL developer at MySQL AB, 2007年4月
MySQL・MariaDB の内部について解説しているおそらく唯一の本です。MariaDB を理解し、開発したい人には必須です。
あらゆる話題を取り扱っているわけではなく、一部は若干時代遅れになっていますが、この項ではまだ最良の書籍です。

MySQL 5.1 Plugin Development (著) Sergei Golubchik and Andrew Hutching, 2010年8月
MariaDB のプラグインを書きたい人は必読です。MySQL と MariaDB のプラグインインターフェースを設計した Sergei 氏の著作です。
