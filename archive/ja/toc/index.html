<<toc>> マクロ

構文 

<<toc [title="目次の見出し"]>>

「title」パラメータは省略可能です。省略した場合は「Contents」になります。

<<toc-item slug="記事URLの末尾#フラグメント識別子" [level=##]>>目次に表示するタイトル<</toc-item>>

省略可能な「level」パラメータには、正の整数を指定します。 このパラメータは複数のインデントレベルを超えることはできませんが、一貫した見出しの生成に役立ちます。

概要

<<toc>> マクロを使って、記事に目次を追加することができます。 目次を配置したい箇所にこのタグを記述します。 目次は記事の見出しをもとに生成されます。

マクロの説明のため、この段落の直後に <<toc>> マクロと、それに続いて見出しをいくつか記述しています。 <<toc-item>> マクロとタイトルについての例は、TOC test ページと TOC Test ページのソース を参照してください。

見出しの ID として利用されるのは、見出しに含まれる英数字のみです。 したがって、英数字を含まない見出しへ目次からリンクするには、styleマクロ を用いて不可視の英数字を追加する必要があります。 例として、この記事のソース を参照してください。
