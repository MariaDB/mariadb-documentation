MariaDB ダウンロード

目次

  1. 最新パッケージ
  2. MariaDB を含んでいるLinuxディストリビューション
  3. リリース前のバイナリ
  4. ソースコードの入手

最新パッケージ

Linux 用 tarball, Windows 用インストーラが mariadb.com/downloads/ もしくは downloads.mariadb.org からダウンロード可能です。

各パッケージングシステムのベストプラクティスを厳守する上で問題がある場合は bug report をお願いいたします。

バイナリパッケージのインストール方法は installing-mariadb-binary-packages から参照可能です。

MariaDB を含んでいるLinuxディストリビューション

  • MariaDBを含んでいるディストリビューション を参照ください。

リリース前のバイナリ

Buildbot システム (Buildbot ページも参照してください) によるバイナリが、http://hasky.askmonty.org/archive/ から取得できます。 実運用システムで使用するのには適していませんが、デバッグには有用でしょう。

上の URLに移動したら、入手したい MariaDB バージョン (5.5 など) とビルド (build-XXXX) を順にクリックします。 ビルド番号 (build-XXXX の XXXX の部分) は、Buildbot の「tarbuildnum」変数と対応しています。

たとえば、MariaDB 5.5 リビジョン 3497 の bsd9-64 ビルドの場合、tarbuildnum は Build Properties に載っている 2434 となります。

ソースコードの入手

ソースコードは https://github.com/MariaDB/server にあります。

MariaDB のローカルブランチを作る方法は、MariaDBのソースコードを入手 ページを参照してください。

MariaDB をソースコードからコンパイルする一般的な方法については、通常のビルド手順 ページを参照してください。
