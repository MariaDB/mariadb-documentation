Livres recommandés sur MariaDB / MySQL

Voici une liste de livres recommandés sur MariaDB et MySQL. Nous avons fourni des liens vers Amazon.com par facilité, mais ils peuvent se trouver chez d'autres librairies, en ligne ou non.

Si vous souhaitez que votre livre préféré sur MySQL / MariaDB soit ajouté ici, merci de laisser un commentaire.

Pour les utilisateurs finaux de MariaDB / MySQL

  • MariaDB Crash Course par Ben Forta
    • Premier livre sur MariaDB !
    • Pour les personnes qui veulent apprendre SQL et les bases de MariaDB.
    • Maintenant disponible. Achetez-le sur Amazon.com ou chez votre libraire préféré.
  • SQL-99 Complete, Really par Peter Gulutzan & Trudy Pelzer.
    • Tout ce que vous vouliez savoir sur la norme SQL 99. Excellent livre de référence !
    • A lire gratuitement dans la base de connaissances !
  • MySQL (4th Edition) par Paul DuBois
    • Le livre 'par défaut' à lire si vous souhaitez apprendre à utiliser MySQL / MariaDB.
  • MySQL Cookbook par Paul DuBois
    • Un tas d'exemples sur comment utiliser MySQL. Comme avec tous les livres de Paul, il vaut son pesant d'or, et même sa lecture est agréable malgré un sujet aussi 'ardu'.
  • High Performance MySQL, Second Edition, Par Baron Schwartz, Peter Zaitsev, Vadim Tkachenko, Jeremy D. Zawodny, Arjen Lentz, Derek J. Balling, et al.
    • "High Performance MySQL is the definitive guide to building fast, reliable systems with MySQL. Written by noted experts with years of real-world experience building very large systems, this book covers every aspect of MySQL performance in detail, and focuses on robustness, security, and data integrity. Learn advanced techniques in depth so you can bring out MySQL's full power." (Extrait de la description du livre chez O'Reilly)
  • MySQL Admin Cookbook
    • Un guide rapide pas-à-pas pour les utilisateurs de MySQL et les administrateurs de bases de données pour s'attaquer aux défis du monde réel avec l'administration et la configuration de MySQL
  • MySQL 5.0 Certification Study Guide, Par Paul DuBois, Stefan Hinz, Carsten Pedersen
    • C'est le guide officiel qui couvre le programme des examens des deux Certifications MySQL. Il est valide jusqu'à la version 5.0 du serveur, et bien qu'il manque toutes les fonctionnalités disponibles dans MySQL versions 5.1 et supérieures (dont MariaDB 5.1 et plus), il apporte une bonne compréhension de base de MySQL pour les utilisateurs finaux.

Pour les développeurs qui veulent coder sur MariaDB ou MySQL

  • Understanding MySQL Internals par Sasha Pachev, ancien développeur MySQL chez MySQL AB.
    • Il s'agit du seul ouvrage à notre connaissance qui décrit le fonctionnement interne de MariaDB / MySQL. Un incontournable pour ceux qui veulent comprendre et développer sur MariaDB !
    • Tous les sujets ne sont pas abordés et quelques parties sont légèrement dépassées, mais il reste le meilleur livre sur ce sujet.
  • MySQL 5.1 Plugin Development par Sergei Golubchik et Andrew Hutchings
    • Un indispensable pour tous ceux qui veulent écrire un plugin pour MariaDB, écrit par le Sergei qui a conçu l'interface des plugins pour MySQL et MariaDB !
