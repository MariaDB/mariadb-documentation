BIGINT

Syntax

BIGINT[(M)] [UNSIGNED] [ZEROFILL]

Description

Un nombre entier étendu. L'intervalle de validité est de -9223372036854775808 à 9223372036854775807. En version non signée (unsigned) l'intervalle est de 0 à 18446744073709551615.

La clause SERIAL est un alias pour: 

BIGINT UNSIGNED NOT NULL AUTO_INCREMENT UNIQUE
