MariaDB c'est quoi?

MariaDB est un remplacement direct et assurant une compatibilité déscendante pour MySQL® Database Server. MariaDB inclus tous les principaux moteurs de stockage de données open source.

Une comparaison de fonctionnalités avec MySQL peut être trouvée ici.

Le code source de MariaDB est disponible publiquement sur Launchpad. Des binaires et paquets sont également disponibles.
