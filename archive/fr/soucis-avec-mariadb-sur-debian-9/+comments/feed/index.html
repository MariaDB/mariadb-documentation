Les deux bases ne peuvent pas coexister sur votre système.</p> <p>Il faudrait donc que vous désinstalliez la base mysql (mysql-server_5.5.9999+default_amd64.deb).</p> <p>Par ailleurs, attention, en supprimant '/var/lib/mysql' vous supprimez les données de votre base de donnée, j'espère que vous avez donc bien une backup (ou export/dump) des vos données quelques part ?</p> <p>Une vois que vous aurez enlevé les paquets relatifs à mysql, sauf les 2 paquets suivants :</p> <pre class="fixed">- mysql-common - libdbd-mysql-perl </pre><p>Essayez de réinstaller mariadb en utilisant la commande suivante :</p> <pre class="fixed">$ sudo apt update $ sudo apt reinstall mariadb-server </pre><p>Cdt, Faustin</p> Faustin Lammlerhttps://mariadb.com/kb/fr/soucis-avec-mariadb-sur-debian-9/+comments/4684
