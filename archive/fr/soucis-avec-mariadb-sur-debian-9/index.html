Bonsoir,

**apt autoclean**

**apt install mariadb-server**

bonjour,

J'ai depuis réglé mon souci ;)

il y avait un souci sur les droit du dossier /tmp

Je l'ai modifié dans my.cnf:

tmpdir = /var/lib/mysql/tmpdir avec les droits mysql:mysql

Puis, j'ai supprimé les deux fichiers de log innodb et hop
