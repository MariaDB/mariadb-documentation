Puis-je obtenir de l'aide sur MariaDB? Quelque chose ne fonctionne pas!

Si vous ne pouvez pas trouver d'aide dans la documentation de MariaDB, dans beaucoup de cas, la documentation de MySQL peut être utilisée. Les fonctionnalités propres à MariaDB sont mentionnées sur la page "La compatibilité entre MariaDB et MySQL" et de manière plus détaillée dans la catégorie MariaDB.

Si vous avez une question à propos d'une fonctionnalité qui n'est pas correctement documentée ou quelque chose qui ne fonctionne pas comme prévu, allez sur la page de la Base de Connaissances correspondant à votre problème et posez-y votre question.

Vous pouvez signaler et vérifier les bugs spécifiques à MariaDB sur JIRA. Les bugs affectant également MySQL doivent également être signalés auprès de MySQL.

Vous pouvez également souscrire aux mailing lists et/ou rejoindre le cannal IRC pour communiquer avec des utilisateurs et développeurs de MariaDB.

La société MariaDB propose un support commercial pour MariaDB et toutes les versions majeures de MySQL (à partir de MySQL 3.23).
