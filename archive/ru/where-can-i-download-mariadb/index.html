Где я могу скачать MariaDB?

Последние версии пакетов

.tar-архивы, бинарные файлы (Linux, Solaris и Windows), а также пакеты для некоторых дистрибутивов Linux доступны по адресу downloads.mariadb.org.

Мы надеемся, что заинтересованные разработчики пакетов из нашего сообщества последуют примеру своих коллег и проявят инициативу в сборке пакетов для своих дистрибутивов. Мы просим Вас придерживаться передовых практик Вашей системы сборки пакетов и приглашаем написать нам отчет об ошибках, если наш проект так или иначе препятствует этому.

Инструкции по установке пакетов можно найти здесь (En).

Бинарные файлы предварительных версий

Бинарные файлы, собранные с помощью нашей системы Buildbot (подробности смотрите на странице "Buildbot" (En)), доступны по адресу http://hasky.askmonty.org/archive/pack/. Они не предназначены для использования в реальных производственных системах, но все же могут быть использованы для отладки.

Перейдя по ссылке выше, кликните по дереву версий MariaDB, которое Вас интересует, а затем выберите номер сборки. Номер сборки совпадает со значением переменной tarbuildnum в Buildbot.

Например, если Вас интересует сборка bsd9-64 из дерева MariaDB 5.5, редакции 3497, то необходимо найти значение tarbuildnum, указанное в таблице "Свойства сборки" ("Build Properties") отчета о сборке Buildbot (En) для данной версии. В данном случае это значение равно "2434".

Исходный код

Вы можете найти весь исходный код по адресу https://github.com/MariaDB/server

Путь наименьшего сопротивления для получения исходного кода - это использование системы контроля исходного кода Git (En). Если Вы незнакомы с git, Вы можете ознакомиться с документацией для лучшего понимания принципов контроля версий с помощью git.

Для того чтобы получить инструкции по созданию локального ответвления MariaDB, ознакомьтесь с информацией на странице "Как получить исходный код MariaDB" (En).

Прочитайте страницу "Общие инструкции по сборке" (En), чтобы получить основные инструкции по компиляции MariaDB из исходников. Страница "Компиляция MariaDB из исходников" (En) содержит информацию об особенностях сборки в зависимости от платформы и дистрибутива, включая информацию о том, как мы выпускаем свои пакеты.

Прежние версии

Лучшим выбором является использование самой свежей версии MariaDB. Однако некоторые организации до сих пор пользуются старой (или очень старой) версией MariaDB/MySQL. Обновление может потребовать внесения значительных изменений в их приложения, в то время как иногда у них даже нет исходного кода этих приложений. Кроме того, операционная система также может быть устаревшей и более не поддерживаемой для MariaDB. Такие компании могут найти очень старые пакеты в репозитории, поддерживаемом MariaDB Corporation, вплоть до очень старых версий, начиная с MySQL 4.1:

https://downloads.mariadb.com/files/
