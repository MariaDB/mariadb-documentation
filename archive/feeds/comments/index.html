E.g.:</p> <pre class="fixed">JSON_SET(config, '$.options', JSON_EXTRACT('{&#34;verbose&#34;:true, &#34;type&#34;: &#34;info&#34;}', '$')) </pre><p>(In MySQL you could <code>CAST(val AS JSON)</code> to do the same thing, but JSON is not a valid target type for CAST in MariaDB.)</p> <p>Note that if you simply use <code>JSON_SET(json_doc, path, '{...}')</code> then you will instead set the value to a single string (containing the JSON-encoded data).</p> <p>Note also that the call to <code>JSON_EXTRACT()</code> must <em>directly</em> provide the value.
