through entire cursor set to avoid erroneous results.</p> <p>Here are 2 examples of trivial aggregate functions that return the "first" item in the aggregate cursor. </p> <p>The first one DECLARE CONTINUE HANDLER FOR NOT FOUND RETURN @iRet; FETCH GROUP NEXT ROW; SET @iRet = reqInt; WHILE TRUE DO FETCH GROUP NEXT ROW; END WHILE; RETURN -1; END // CREATE OR REPLACE DEFINER=`root`@`localhost` AGGREGATE FUNCTION `aggFirst_int2`(`reqInt` INT) RETURNS int(11) BEGIN DECLARE CONTINUE HANDLER FOR NOT FOUND RETURN @iRet; FETCH GROUP NEXT ROW; SET @iRet = reqInt; RETURN @iRet; END // DELIMITER ; SELECT MAX(fInt), MIN(fInt), COUNT(fInt), aggFirst_int(fInt), aggFirst_int2(fInt), fChar FROM (SELECT "a" AS fChar, 1 AS fInt UNION SELECT "b", 4 UNION SELECT "b", 2 UNION
