Estilos de tablas HTML

Las tablas sin estilo son como esta:

Column OneColumn TwoColumn ThreeColumn Four
FruitApplesOrangesGrapes
EntertainmentBooksRadioMovies
ExerciseRunningWalkingJumping

Para definir un estilo en una tabla incrústala entre etiquetas de estilo como esta:

<<style class="table-class">>
...la tabla va aquí...
<</style>>

Las siguientes clases de tablas están disponibles:
