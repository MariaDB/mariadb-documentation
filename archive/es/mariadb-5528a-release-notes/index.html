MariaDB 5.5.28a Release Notes

Download | Release Notes | Changelog

Release date: 29 Nov 2012

MariaDB 5.5.28a is a Stable (GA)

Lanzamiento. en general, esto significa que no hay bugs conocidos graves, a excepción de los marcados como solicitudes de características, que no había bichos fueron corregidos desde el ultimo lanzamiento que causó notables cambios en el código, y creemos que el código está listo para su uso general (basada en flujo bug).

Para una descripcion de MariaDB 5.5 ver el What is MariaDB 5.5 page.

Para obtener una lista de los cambios realizados en MariaDB 5.5.28a, con enlaces a la información detallada, ver el enlace MariaDB 5.5.28a Changelog.

En muchos aspectos MariaDB wfuncionan exactamente como MySQL: todos los comandos, interfaces, APIs y bibliotecas que existen en MySQL también existen en MariaDB.

Importante correccion de seguridad para Bug desbordamiento de Buffer

MariaDB 5.5.28a incluye una solución para CVE-2012-5579611, una vulnerabilidad que permite a un usuario autenticado se bloquee servidor MariaDB o para ejecutar código arbitrario con los privilegios del proceso mysqld. Este es un problema de seguridad grave. Te recomendamos actualizar desde versiones anteriores tan pronto como sea posible.

Gracias, y disfrute de MariaDB!

Ser notificado de las nuevas versiones automáticamente agregar a releases rss feed to su lector de RSS favorito o por subscribing lista de anuncios anuncio mariadb.org (esto es un poco tráfico, sólo para anunciar la lista).

MariaDB puede estar ya incluido en su distribución favorita OS. Más información se puede encontrar en las distribuciones que incluyen la página MariaDB.
