¿Qué es MariaDB 10.0?

MariaDB 10.0 es la versión de desarrollo de MariaDB. Esta construida sobre la serie de MariaDB 5.5 con caracteristicas portadas de MYSQL 5.6 y nuevas características que no se encuentran en ningún otro lado.

Blog posts with details of the reasoning behind calling this version MariaDB 10:

  • http://blog.mariadb.org/mariadb-10-0-and-mysql-5-6/
  • http://blog.mariadb.org/what-comes-in-between-mariadb-now-and-mysql-5-6/
  • http://blog.mariadb.org/explanation-on-mariadb-10-0/

Caracteristicas Implementadas

Características que son liberadas, o en el fuente sobre la plataforma de lanzamiento.

Nuevas Caracteristicas

  • Transacción Global ID (MDEV-26)
  • Replicación de Origen Multiple (MDEV-253) — Código original de Taobao, developed by Peng Lixun.
  • Motor de Almacenamiento Cassandra (MDEV-4695)
  • Motor de Almacenamiento CONNECT (MDEV-4146)
  • Mejores Tablas de Desubrimiento. Motor de Almacenamiento Sequence. Asistencia de descubrimineto en FederatedX (MDEV-3808)
  • Motor de Almacenamiento Spider (MDEV-4438)
  • Motor de Almacenamiento TokuDB (MDEV-4507)
  • Motor de estadísticas de tabla independientes (MDEV-3806)
  • Optimización de sub-consultas: optimización de EXISTS-to-IN, (MDEV-38 - NOT EXISTS to IN), MDEV-537, MDEV-3862
  • Histogramas basados en estadísticas para columnas no indexadas (MDEV-4145)
  • Comando SHOW EXPLAIN (MDEV-165)
  • EXPLAIN en el registro de consultas lentas (explain-in-the-slow-query-log) (MDEV-407)
  • |Por el uso de memoria (MDEV-4011). — Código original de Taobao, developed by Peng Lixun.
    • information_schema.processlist tiene dos nuevas columnas: MEMORY_USAGE y EXAMINED_ROWS.
    • SHOW STATUS tiene una nueva variable: Memory_used.
  • SHOW PLUGINS SONAME 'XXX' (MDEV-3807)
  • Sentencia SHUTDOWN (MDEV-4660)
  • Eliminar una consulta por el id de una consulta, no por hilo de identificación (MDEV-4911)
  • Rápida generación de llaves UNIQUE con ALTER TABLE (MDEV-539)
  • Implement async commit checkpoint in InnoDB and XtraDB (MDEV-532)
  • Soporte para atomic writes on FusionIO DirectFS (MDEV-4338)
  • DELETE ... RETURNING (MDEV-3814)
  • IF (NOT) EXIST clausas para ALTER TABLE (MDEV-318)
  • Columnas dinámicas ahora soporta nombres (MDEV-377, [[dynamic-columns-in-mariadb-10|Resumen de Cambios])
  • Múltiples usos de bloqueos (GET_LOCK) en una conexión (MDEV-3917)
  • Mejores mensajes de error (Todos los números de error ahora incluyen texto descriptivo de lo que significa el numero)
  • Atributos de tabla con sysvar como un valor por defecto (MDEV-4022)
  • Mejoras en expresiones regulares
    • Nueva libreria de expresiones regulares con modernas caracteristicas (PCRE)
    • Nuevas funciones REGEXP_REPLACE, REGEXP_INSTR, REGEXP_SUBSTR.
  • Roles (MDEV-4397)

Nuevas características re-implementadas desde una caracteristica similiar de MYSQL

  • CURRENT_TIMESTAMP como DEFAULT for DATETIME columnas (MDEV-452)
  • EXPLAIN para INSERT/UPDATE/DELETE (MDEV-3798, MWL#51)

Nuevas características portadas desde MySQL 5.6

  • Nuevo InnoDB — desde MySQL 5.6.10 en MariaDB 10.0.4 en adelante
  • Esquema de rendimiento — desde MySQL 5.6.10 en MariaDB 10.0.4 en adelante
  • Transacción optimizado de solo lectura (para InnoDB). Esto incluye soporte para TRANSACTION READ ONLY.
  • Optimización de organizaión de archivos para consultas usando ORDER BY ... LIMIT optimización - Una optimización útil para mostrar sólo un par de filas de un resultado mayor. (MDEV-4026)
  • Portado --plugin-load-add (MDEV-3860)
  • ALTER TABLE en línea (MDEV-3933)
  • Privilegios en tablas temporales
  • Extensiones de caracteres seleccionados
  • GET DIAGNOSTICS
  • Literales temporales (tal como TIME'12:34:56')
