La macro <<fixed>>

Es igual que <<code>> pero con ventajas en ciertas situaciones. Muestra el texto con una fuente de ancho fijo.

<<fixed>>Blah blah blah<</fixed>>

blah blah blah

<<fixed>> blocks can contain things which <<code>> blocks can't:

<<fixed>>
* dos
* listas

y un [[mariadb|enlace]]

# dentro
# de un <<fixed>>
# bloque
<</fixed>>

<<code>>
* dos
* listas

y un [[mariadb|enlace]]

# dentro
# de un <<fixed>>
# bloque
<</code>>
