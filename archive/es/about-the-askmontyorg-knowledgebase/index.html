Acerca del Banco de Conocimientos de Askmonty.org

El Banco de Conocimientos (Knowledgebase o KB) de AskMonty es un depósito para toda la información relacionada con MariaDB así como de información sobre MySQL®. Cualquiera puede comenzar contribuyendo ya mismo bien Registrándose con una nueva cuenta, bien logándose con una existente OpenID.Una vez que estés logado, puedes hacer preguntas, editar páginas o incluso crear páginas nuevas. Lee la sección Cómo funciona (How it works) para comenzar.

Propósito

El Banco de Conocimientos (KB) de AskMonty existe para documentar y responder preguntas frecuentes (FAQs) acerca de MariaDB y MySQL.

El Banco de Conocimientos sirve como documentación básica para la base de datos de código abierto MariaDB. Cuando es posible, todo el contenido tiene una doble licencia: bajo la cláusula de Creative Commons de Atribución - Compartir Igual, y la GFDL (Licencia de Documentación Libre de GNU). Algún contenido (por ejemplo, el código fuente de MariaDB) se utiliza bajo licencia de GPLv2.[1]

El Banco sigue una estructura arborescente. Cuando un/a usuari@ tiene una pregunta, se le anima a navegar a través del Banco de Conocimientos hasta donde cree que pueda estar la respuesta. Si la respuesta no está ahí, el/la usuari@ puede realizar la pregunta desde esa página. Si la respuesta existe en alguna otra parte en el Banco, la misma es enlazada a la página a la que fue el/la usuari@. Si la respuesta no existe, miembros de la Comunidad MariaDB la responden. La respuesta a la pregunta del/la usuari@ estará entonces en un buen (y esperemos que lógico) lugar, lista para la próxima vez que un/a usuari@ visite el Banco de Conocimientos con la misma pregunta.

La meta es que sea un sustituto a los Foros, listas de emails (cuando se pida ayuda), un sistema de localización de errores, FAQ y un manual de MySQL/MariaDB.

Historia

¿Por qué se llama el Banco de Conocimientos AskMonty? De forma simplificada, es porque el creador de la base de datos MySQL es Michael Widenius, y su apodo durante mucho tiempo ha sido Monty. Así que el Banco de Conocimientos es una forma de preguntar a Monty cuestiones y conseguir respuestas. Monty pasa la mayor parte de su tiempo desarrollando MariaDB, pero quita tiempo de su ocupada agenda para responder preguntas y escribir artículos aquí. Otros desarrolladores también contribuyen al Banco de Conocimientos de forma habitual.

Implementación

El KB o Banco de Conocimientos está escrito en Django con un backend o soporte de MariaDB. El código será publicado como GPL tan pronto como sea estable.

Contribución

La páginaContribuyendo al Banco de Conocimientos AskMonty (KB) proporciona información acerca de cómo puedes ayudar a mejorar este Banco de Conocimientos.

Las estadísticas de contribución de todos los usuarios pueden ser consultadas aquí: http://kb.askmonty.org/stats/users/
