MariaDB Knowledge Base Comments for: Subquery and EXISTS issuehttps://mariadb.com/kb/en/subquery-and-exists-issue/+comments/feed/en-usMon, 21 Apr 2025 10:58:53 +0000Re: Subquery and EXISTS issuehttps://mariadb.com/kb/en/subquery-and-exists-issue/+comments/6896<p>If you're trying to get the first query to return the same result as the second query, this is not how EXISTS works. If the subquery returns any rows at all, then the subquery returns true (and therefore the query is effectively <code>Select e1.name from employee e1</code>), otherwise it returns false. See <a href="/kb/en/subqueries-and-exists/">Subqueries and EXISTS</a>.</p> Ian Gilfillanhttps://mariadb.com/kb/en/subquery-and-exists-issue/+comments/6896
