Plans for MariaDB 11.3

Contents

  1. JIRA
  2. Contributing
  3. See Also

MariaDB 11.3 is stable, so no new features will be added. See Plans for MariaDB 11.5 instead.

JIRA

We manage our development plans in JIRA, so the definitive list will be there. This search shows what we currently plan for 11.3. It shows all tasks with the Fix-Version being 11.3. Not all these tasks will really end up in 11.3 but tasks with the "red" priorities have a much higher chance of being done in time for 11.3. Practically, you can think of these tasks as "features that will be in 11.3". Tasks with the "green" priorities probably won't be in 11.3. Think of them as "bonus features that would be nice to have in 11.3".

Contributing

If you want to be part of developing any of these features, see Contributing to the MariaDB Project. You can also add new features to this list or to JIRA.
