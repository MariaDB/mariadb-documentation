MariaDB 10.1.16 Changelog

The most recent release of MariaDB 10.1 is:
MariaDB 10.1.48 Stable (GA) Download Now

Download Release Notes Changelog Overview of 10.1

Release date: 18 Jul 2016

For the highlights of this release, see the release notes.

This changelog only details pushes made between 10.1.15 and 10.1.16. For changes made between 10.1.14 and 10.1.15, see the unreleased MariaDB 10.1.15 Changelog
