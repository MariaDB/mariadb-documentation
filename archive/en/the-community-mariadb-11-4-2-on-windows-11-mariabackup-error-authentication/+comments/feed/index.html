You can't load client plugin into server.</p> <p>You just need plugin-dir under [client], usually. But mariabackup is slightly (or not so slightly) weird, it uses the server's builtin client, so there can be problems.</p> <p>An educated guess is that you simply mistype the password, therefore server tries to use another plugin, which is gssapi, which then fails on reasons we do not know of yet.</p> <p>Can also share you my.ini? Best thing you can do is to create a bug at <a href="https://jira.mariadb.org/">https://jira.mariadb.org/</a> , with detailed "how to reproduce". I gave it a try, and it worked for me, but maybe you installed it differently somehow.</p> Vladislav Vaintroubhttps://mariadb.com/kb/en/the-community-mariadb-11-4-2-on-windows-11-mariabackup-error-authentication/+comments/6836
