MariaDB Knowledge Base Comments for: Portable MariaDBhttps://mariadb.com/kb/en/portable-mariadb/+comments/feed/en-usTue, 14 Nov 2023 00:03:27 +0000Re: Portable MariaDBhttps://mariadb.com/kb/en/portable-mariadb/+comments/480<p>You can use a binary tarball distribution (or zip, if you are on Windows), e.g. <a href="https://downloads.mariadb.org/mariadb/5.5.29/#os_group=linux_generic&amp;os_group=windows&amp;file_type=tar_gz&amp;file_type=zip">binary tarballs and zip files of MariaDB 5.5.29</a>. All you need to do is download an archive onto your USB drive, unpack there, and as long as you run the server from inside the unpacked directory with --no-defaults option, all paths will be relative and the data directory will be also inside.</p> Elena Stepanovahttps://mariadb.com/kb/en/portable-mariadb/+comments/480
