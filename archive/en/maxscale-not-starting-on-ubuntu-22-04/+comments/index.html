1 year, 11 months ago Markus Mäkelä
Re: Maxscale not starting on Ubuntu 22.04

I think this might be a problem that's filed as MXS-4599. MaxScale ends up using instructions that aren't allowed on older CPUs that don't support the AVX2 instruction set. This should be fixed in the upcoming MaxScale releases.

If you can enable AVX2 instructions on that virtual machine, it should allow MaxScale to start correctly even with an older version.
