MariaDB 11.7 is an upcoming release series.

Upgrading

  • See Upgrading Between Major MariaDB Versions and Upgrading from MariaDB 10.11 to MariaDB 11.0.

New Features

  • Async rollback prepared transactions during binlog crash recovery (MDEV-33853)

Catalogs

  • MariaDB catalogs are a multi-tenancy feature where a single instance MariaDB server handles multiple independent tenants (customers), who have their own users, schemas etc. See MDEV-31542 "Add multi-tenancy catalogs to MariaDB" for details.

Variables

Deprecated Variables

  • binlog_optimize_thread_scheduling (MDEV-33756)
  • The Spider variables spider_table_crd_thread_count and spider_table_sts_thread_count (MDEV-28009)

Security Vulnerabilities Fixed in MariaDB 11.7

For a complete list of security vulnerabilities (CVEs) fixed across all versions of MariaDB, see the Security Vulnerabilities Fixed in MariaDB page.
