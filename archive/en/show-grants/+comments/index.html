Re: SHOW GRANTS

The syntax section should say:

SHOW GRANTS [FOR user|role]

adding "|role" cause this command accepts a role as subject. This means you can show the grants for an specified role and not only for a user, as the syntax suggest (as is at 11/26/2020)
