Re: Upgrade from 10.5.x to 10.6.x mysql_upgrade or mariadb-upgrade not working

The mariadb-upgrade in 10.6 need to have a version that strictly matches the server version. If you have a lightly different client and server package this might be the error. Also if you force the server version you may see differences.

The mysql_upgrade and mariadb-upgrade are the same program, we are just in the process of removing mysql names slowly.

The exact error message and the mariadb-upgrade and mariadbd (server) versions would help.
