commands we use to create our Debian packages (starting with a download of the source tarball, I'm using <a href="https://downloads.mariadb.org/mariadb/5.5.32/#os_group=source">5.5.32</a> in my example).</p> <pre class="fixed">mkdir -p debs/binary debs/source mkdir build <span class="nb">cd</span> build tar zxf <span class="s2">&quot;../mariadb-5.5.32.tar.gz&quot;</span> <span class="nb">cd</span> <span class="s2">&quot;mariadb-5.5.32&quot;</span> <span class="nv">AM_EXTRA_MAKEFLAGS</span><span class="o">=</span><span class="nv">VERBOSE</span><span class="o">=</span><span class="m">1</span> debian/autobake-deb.sh cp <span class="sb">`</span>find .. -maxdepth <span class="m">1</span> -type f<span class="sb">`</span> ../../debs/binary/ <span class="nb">cd</span> ../../debs dpkg-scanpackages binary /dev/null <span class="p">|</span> gzip -9c &gt; binary/Packages.gz dpkg-scansources <span class="nb">source</span> /dev/null <span class="p">|</span> gzip -9c &gt; source/Sources.gz </pre><p>You'll need the dpkg-dev package installed, possibly others.</p> <p>Thanks.</p> Daniel
