MariaDB Knowledge Base Comments for: Moving from MariaDB 10.2 to 10.11 on Windows OShttps://mariadb.com/kb/en/moving-from-mariadb-10-2-to-10-11-on-windows-os/+comments/feed/en-usTue, 10 Oct 2023 01:05:00 +0000Re: Moving from MariaDB 10.2 to 10.11 on Windows OShttps://mariadb.com/kb/en/moving-from-mariadb-10-2-to-10-11-on-windows-os/+comments/6360<p>Provided the 10.11 service on Windows Server 2022 is stopped, copying all the files from the stopped 10.2 service on Windows 2012 is a good way to upgrade.</p> <p>Before copying ensure that the shutdown on 10.2 was clean in the Windows Event Log and didn't end in a crash and ends in "Shutdown complete".</p> <p>No need to upgrade the old version first.</p> <p>Run <a href="/kb/en/mariadb-upgrade/">mariadb-upgrade</a> after the 10.11 service is started.</p> Daniel Blackhttps://mariadb.com/kb/en/moving-from-mariadb-10-2-to-10-11-on-windows-os/+comments/6360
