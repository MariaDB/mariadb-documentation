Re: Transfer of a website on a MySQL8.0.27 server onto a MariaDB 10.4 server

There's nothing completely automated, but you would probably want to use mysqldump to transfer the data, and check for incompatibilities. See Upgrading from MySQL to MariaDB and Incompatibilities and Feature Differences Between MariaDB 10.4 and MySQL 8.0. The solution depends on what the incompatibilities are.
