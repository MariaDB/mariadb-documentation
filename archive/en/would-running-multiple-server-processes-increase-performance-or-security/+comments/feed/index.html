MariaDB is very good and ensuring a fair use of resources between multiple connections however inadequite tuning/monitoring of connections/table cache elements can cause some contention between different applications. Running separate instances will still require these to be tuned per application.</p> <p>MariaDB access control with a user per db application (with limited grants) does a very good job of security for many applications.
