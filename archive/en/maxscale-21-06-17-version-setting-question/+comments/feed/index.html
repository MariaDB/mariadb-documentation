MariaDB Knowledge Base Comments for: maxscale 21.06.17 version setting questionhttps://mariadb.com/kb/en/maxscale-21-06-17-version-setting-question/+comments/feed/en-usMon, 12 May 2025 08:04:05 +0000Re: maxscale 21.06.17 version setting questionhttps://mariadb.com/kb/en/maxscale-21-06-17-version-setting-question/+comments/6900<p>Weights have been removed from MaxScale but you can usually achieve a better result by customizing the routing with the <a href="https://mariadb.com/kb/en/mariadb-maxscale-2106-maxscale-2106-named-server-filter/">namedserverfilter</a>.</p> <p>As for maintenance mode causing applications to close, this is expected behavior: servers in maintenance do not interrupt ongoing queries but new queries on them are refused. A less strict version of this is the drain state that can be set with <code>maxctrl set server &lt;server&gt; drain</code>. Draining servers prevents new connections from being created but it lets existing connection to continue using them.</p> Markus Mäkelähttps://mariadb.com/kb/en/maxscale-21-06-17-version-setting-question/+comments/6900
