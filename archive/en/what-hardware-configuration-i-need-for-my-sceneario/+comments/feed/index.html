MariaDB Knowledge Base Comments for: What hardware configuration I need for my sceneario?https://mariadb.com/kb/en/what-hardware-configuration-i-need-for-my-sceneario/+comments/feed/en-usMon, 09 Sep 2024 13:14:24 +0000Re: What hardware configuration I need for my sceneario?https://mariadb.com/kb/en/what-hardware-configuration-i-need-for-my-sceneario/+comments/4724<p>There is no difference between one server and multiple for the same resources. Databases within a server are quite de-conflicted from each other.</p> <p>Cases for multiple servers (or instances within the same server) include security isolation (beyond user level privs). Global tunings required to be different per database. Different MariaDB versions required per database.</p> <p>A single instances is a good way to start, and if any of the above cases arise a particular database can be split off to a new instance.</p> <p>Hardware specifications will vary considerably depending on what sorts of queries you are doing.</p> <p>With your existing databases I assume you have some measure of how much resources each use already?</p> Daniel Blackhttps://mariadb.com/kb/en/what-hardware-configuration-i-need-for-my-sceneario/+comments/4724
