But if you <a href="http://downloads.askmonty.org/mariadb/repositories/">upgrade</a> to <a href="/kb/en/what-is-mariadb-52/">MariaDB 5.2</a> or 5.3 the error will go away because apt won't see a MySQL 5.1.61 update as belonging to <a href="/kb/en/what-is-mariadb-52/">MariaDB 5.2</a>.x.</p> <p>I've updated several servers from MariaDB and MySQL 5.1.x to <a href="/kb/en/what-is-mariadb-52/">MariaDB 5.2</a>.x and 5.3.x and never had any trouble.</p> Daniel Bartholomewhttps://mariadb.com/kb/en/ubuntu-1004-lts-server-now-a-package-conflict-with-libmysqlclient16/+comments/275
