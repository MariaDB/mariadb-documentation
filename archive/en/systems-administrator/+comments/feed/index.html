MariaDB Knowledge Base Comments for: Systems Administratorhttps://mariadb.com/kb/en/systems-administrator/+comments/feed/en-usTue, 12 Mar 2024 08:13:14 +0000Re: Systems Administratorhttps://mariadb.com/kb/en/systems-administrator/+comments/5717<p>MaxScale has no minimum required amount of memory but a good starting point is to take half of the resources a single MariaDB node is using. For example, if the database servers have 8 cores with 16GB of RAM, a MaxScale server should have 4 cores and 8GB of RAM.</p> <p>This is just a starting point and you'll need to estimate how much resources you need. The actual amount of resources needed depends on the modules you use and the traffic that goes through MaxScale. If you use modules like the cache filter, you'll benefit from having more RAM on the MaxScale server compared to a configuration where you don't use it.</p> Markus Mäkelähttps://mariadb.com/kb/en/systems-administrator/+comments/5717
