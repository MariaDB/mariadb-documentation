MariaDB Knowledge Base Comments for: STDDEV_SAMPhttps://mariadb.com/kb/en/stddev_samp/+comments/feed/en-usMon, 27 Jan 2025 03:56:08 +0000Re: STDDEV_SAMPhttps://mariadb.com/kb/en/stddev_samp/+comments/5011<p>Beware using this function in a view under <a href="/kb/en/what-is-mariadb-102/">MariaDB 10.2</a> up to latest version.</p> <p>When creating views, those versions erroneously convert any call to STDDEV_SAMP to STD (a shortcut to STDDEV_POP). Since those functions do not behave the same, you will likely have incorrect calculations in those views.</p> <p>There is a workaround: use SQRT(VAR_SAMP(expr)) instead of STDDEV_SAMP(expr).</p> <p>Reference: <a href="https://jira.mariadb.org/browse/MDEV-19071">https://jira.mariadb.org/browse/MDEV-19071</a></p> <p>GR.</p> Guy Rouillardhttps://mariadb.com/kb/en/stddev_samp/+comments/5011
