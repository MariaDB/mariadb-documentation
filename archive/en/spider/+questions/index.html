Spider

You are viewing questions related to Spider.

The Spider storage engine is a federated database solution that supports partitioning and xa transactions, and allows tables of different MariaDB instances to be handled as if they were on the same instance.

Versions of Spider in MariaDB

From MariaDB 10.9.2, the Spider version number matches the server version.
