MariaDB Knowledge Base Comments for: Uninstall mariadb on OSX?https://mariadb.com/kb/en/uninstall-mariadb-on-osx/+comments/feed/en-usTue, 27 Feb 2024 04:36:55 +0000Re: Uninstall mariadb on OSX?https://mariadb.com/kb/en/uninstall-mariadb-on-osx/+comments/520<p>If you compiled and then used '<code>sudo make install</code>' to install <a href="/kb/en/what-is-mariadb-51/">MariaDB 5.1</a>.x, <a href="/kb/en/what-is-mariadb-52/">MariaDB 5.2</a>.x, or <a href="/kb/en/what-is-mariadb-53/">MariaDB 5.3</a>.x then you can uninstall MariaDB with:</p> <pre class="fixed">sudo make uninstall </pre><p>If you compiled and installed <a href="/kb/en/what-is-mariadb-55/">MariaDB 5.5</a>.x or higher then there is no '<code>make uninstall</code>'. However, after doing a 'sudo make install' to install MariaDB there will be an '<code>install_manifest.txt</code>' file at the top level of the source dir. Remove all of the files listed in it and everything you installed will be gone.</p> <p>On Linux the default is to place everything under '<code>/usr/local/mysql/</code>' so removing that directory is enough to uninstall MariaDB. If you customized the install directory, or if MacOS has a different default then it will be different. In any case the '<code>install_manifest.txt</code>' file will have the locations.</p> Daniel Bartholomewhttps://mariadb.com/kb/en/uninstall-mariadb-on-osx/+comments/520
