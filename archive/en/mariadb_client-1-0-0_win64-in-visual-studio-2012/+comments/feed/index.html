This can be set under the Property Sheet of the Project. On the very top of the windows displaying the properties, the user can see "win32" environment (even though the Visual Studio 2012 can do x64 and was installed under Win7 x64). opening the list of choices does NOT offer x64, but just new... The user must chose "new..." environment which will then offer "x64" as a possible new choice.
