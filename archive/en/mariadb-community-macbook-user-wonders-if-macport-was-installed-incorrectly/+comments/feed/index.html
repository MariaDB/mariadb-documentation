If it got as far as installing a launchd script, then you could end up with two copies of MariaDB running, competing for the same default port (3306).</p> <p>I think the Homebrew folks recommend uninstalling other package managers before installing Homebrew, as sometimes they fight over the $PATH variable.</p> <p>So, first do something like "port uninstall mariadb" (but look it up, I don't have it installed, so I can't tell you how it works).</p> <p>Then use the MacPorts procedure for uninstalling itself.</p> <p>Then install Homebrew.</p> <p>Then do "brew install mariadb".
