MariaDB Enterprise Server 5.5.50 Release Notes

This version of MariaDB Enterprise Server is built off MariaDB 5.5.50.

MariaDB Enterprise Cluster 5.5.50 Release Notes

This version of MariaDB Enterprise Cluster is built off MariaDB Galera Cluster 5.5.50 and is mainly an upgrade to benefit from many fixes and improvements in MariaDB Galera 5.5.50, including some backported fixes, as detailed in the changelog.
