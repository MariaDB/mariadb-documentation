MariaDB Knowledge Base Comments for: Using traditional UNIX log locationshttps://mariadb.com/kb/en/using-traditional-unix-log-locations/+comments/feed/en-usMon, 23 Sep 2024 14:07:59 +0000Re: Using traditional UNIX log locationshttps://mariadb.com/kb/en/using-traditional-unix-log-locations/+comments/1284<p>There is a typo in your example above, it should be --log-error=xxx, not --log=err=xxx, if that was what you entered, it wouldn't have worked.</p> <p>Either way, in Ubuntu, the error log goes to syslog due to /etc/mysql/conf.d/mysqld_safe_syslog.cnf</p> <p>There should be a note about this in your my.cnf config file, and you can set it there <code>log-error = xxx</code>, and if needs be comment out the syslog setting.</p> Ian Gilfillanhttps://mariadb.com/kb/en/using-traditional-unix-log-locations/+comments/1284
