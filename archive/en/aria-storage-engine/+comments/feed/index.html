MariaDB Knowledge Base Comments for: Aria Storage Enginehttps://mariadb.com/kb/en/aria-storage-engine/+comments/feed/en-usMon, 30 Dec 2024 05:02:37 +0000Re: Aria storage Enginehttps://mariadb.com/kb/en/aria-storage-engine/+comments/11<p>You will need to update your tables to use the Aria storage engine.</p> <p>Something like this:</p> <p>ALTER TABLE `tablename` ENGINE=`MARIA` TRANSACTIONAL=1;</p> <p> <em>The Aria storage engine is still called 'Maria' in <a href="/kb/en/mariadb-5149-release-notes/">MariaDB 5.1.49</a>.</em></p> <p>This is not something you want to do on a large table or busy production server without bringing the server down first. Rebuilding a table can take a long time (depending on the size) and the table will be inaccessible while the change is being made.</p> <p>As with any change, test it on a non-production table with example data first and test to ensure your data and applications work fine after the change has been made.
