In MariaDB, the <code>localhost</code> address is actually referring to the local UNIX domain socket (usually at <code>/var/lib/mysql/mysql.sock</code>) that the server listens on and is separate from the address defined by bind_address.</p> <p>If possible, creating a completely new user guarantees that no default configurations are in use.
