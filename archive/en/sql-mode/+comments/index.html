5 years, 2 months ago Roger Morgan
Re: SQL_MODE

SQL_MODE is not mainly "for getting MariaDB to emulate behavior from other SQL servers", it's for getting MariaDB to conform to the SQL standard by specifying SQL_MODE = ANSI .

A side effect is that complying with standards makes it easier to port one's work to a different database, in case that's ever needed - but is it really necessary to explain why conformance with standards is a good thing?
