When running the mariadb_repo_setup script, you can tell it what OS and version to use and then it will skip trying to detect the OS you are running:

./mariadb_repo_setup --os-type=rhel --os-version=8
 
2 years, 4 months ago Daniel Black
Re: No Rocky-Linux support yet??

Hi, I've created MDEV-25805 to add support into the maradb_repo_setup script.

Supporting yet another Linux is always a big step that consumes a lot of packaging/CI time. At the moment we've left RHEL compatible Linuxs to be work with just the RHEL packages. This includes even currently reasonably popular Linuxs like Oracle Linux and CloudLinux.

Rocky Linux, AlmaLinux and the like going to have to maintain enough compatibility to make this true however if its users see a minor tweak in packaging required to make this happen we welcome a bug report.
