To enable it, add the line <code class="fixed" style="white-space:pre-wrap">&#34;show_compatibility_56=ON&#34;</code> under <code class="fixed" style="white-space:pre-wrap">[mysqld]</code> in MySQL configuration file and restart the MySQL server.</p> <p>Also, we did fix this issue in Monyog 8.0.3, such that "Slave_running?" counter will display "(n/a)" instead of "No", if the "show_compatibility_56" MySQL variable is OFF. Note that this is a temporary fix/workaround for false alerts, we will be fixing it in one of our coming releases.
