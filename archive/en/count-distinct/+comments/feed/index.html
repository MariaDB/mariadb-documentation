MariaDB Knowledge Base Comments for: COUNT DISTINCThttps://mariadb.com/kb/en/count-distinct/+comments/feed/en-usMon, 26 May 2025 03:03:14 +0000Re: COUNT DISTINCThttps://mariadb.com/kb/en/count-distinct/+comments/5806<p>Question: Is COUNT DISTINCT optimized to take advantage of sorted values?</p> <p>For example, if I have a sorted column <code>LAST_NAME</code>, then <code>SELECT COUNT(DISTINCT(LAST_NAME)) FROM PEOPLE</code> can be performed in a constant amount of memory: if you iterate the rows in <code>LAST_NAME</code> order, then you're guaranteed that all distinct last name values appear contiguously, so there's no need to remember which one's you've already seen.</p> Archie Cobbshttps://mariadb.com/kb/en/count-distinct/+comments/5806
