MariaDB Connector/J 1.3.0 Changelog

The most recent Stable (GA) release of MariaDB Connector/J is:
MariaDB Connector/J 3.1.4

Download Release Notes Changelog Connector/J Overview

Release date: 16 Nov 2015

MariaDB Connector/J 1.3.0 is a Stable (GA) release.

For the highlights of this release, see the release notes.

The revision number links will take you to the revision's page on GitHub. On GitHub you can view more details of the revision and view diffs of the code modified in that revision.

Major change :

  • CONJ-94 : new Date/Time/Timstamps implementation that handle timezone and daylight correctly.
    • Revision #b186cc5 2015-09-17
    • Revision #27d41bf 2015-09-21
    • Revision #2619ff5 2015-09-29
  • CONJ-189 : implementation of loadbalancing without failover
    • Revision #8e05ce6 2015-09-17
    • Revision #c6150f6 2015-09-21
  • CONJ-22 : Implementation of prepareStatement server side
    • Revision #9c0389b 2015-09-08
    • Revision #c04e4e3 2015-09-10
    • Revision #fb995b8 2015-09-15
    • Revision #0f78c03 2015-09-15
    • Revision #b300e40 2015-09-15

Minor change :

  • CONJ-177 : new option (assureReadonly) to validate that slave is read-only - Revision #02449e8 2015-09-21
  • CONJ-200 : fixing insert ... select error in prepare statement
  • CONJ-192 : Correct driver metadata version - Revision #40a1c83 2015-09-04
  • CONJ-185 : Handle setObject java.lang.Character type - Revision #9f3eb20 2015-09-15
  • CONJ-182 : Correction of "Self assignment of field correction - Revision #c317981 2015-09-15

MISC :
