To configure it, you can use <a href="/kb/en/multi-source-replication/">multi-source-replication</a>.</p> <p>It can work as long as you guarantee that no two writes can ever conflict between Server 1 and Server 3. This is usually quite hard to guarantee if you intend the same applications to write to both Server1 and Server 3.
