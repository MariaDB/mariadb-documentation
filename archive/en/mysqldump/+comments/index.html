We were seeing some cases on SimpleBackups where an issue related to column statistics when running mysqldump used to show up. If this is the case be sure to confirm that you are not using MySQL 8 tools.

The flag column-statistics is only supported by MySQL 8's mysqldump and not MariaDB.
