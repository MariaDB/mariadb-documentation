Here you can see an example in Go: <a href="http://databaseblog.myname.nl/2015/10/a-mysql-udf-written-in-go.html">http://databaseblog.myname.nl/2015/10/a-mysql-udf-written-in-go.html</a></p> <p>Stored procedures, functions and triggers must be written in procedural SQL.</p> <p>-As the Oracle package doesn't exist, how MariaDB structure its code ?</p> <p>Stored procedures are stored in a database, you can see the database as a namespace. There is no package.</p> <p>-Is there a plugin which can call a Java proc in MariaDB , other than the UDF function (as MariaDB cannot call directly a java proc)?</p> <p>No, currently there is no way to write a procedure in an external language.
