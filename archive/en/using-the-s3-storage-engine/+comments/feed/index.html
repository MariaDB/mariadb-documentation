MariaDB Knowledge Base Comments for: Using the S3 Storage Enginehttps://mariadb.com/kb/en/using-the-s3-storage-engine/+comments/feed/en-usMon, 02 Oct 2023 18:47:00 +0000Re: Using the S3 Storage Enginehttps://mariadb.com/kb/en/using-the-s3-storage-engine/+comments/4779<p>'Troubleshooting S3 on SELinux': disabling SELinux is generally not advisable and it is possible to allow MariaDB to connect to ports other than 3306</p> <p>Search for the ports allowed for MariaDB:</p> <pre class="fixed">$ sudo semanage port -l | grep mysqd_port_t mysqld_port_t tcp 1186, 3306, 63132-63164 </pre><p>Say you want to allow MariaDB to connect to port 32768:</p> <pre class="fixed">$ sudo semanage port -a -t mysqld_port_t -p tcp 32768 </pre><p>Search again for the ports allowed for MariaDB:</p> <pre class="fixed">$ sudo semanage port -l | grep mysqd_port_t mysqld_port_t tcp 32768,1186, 3306, 63132-63164 </pre><p>Connecting to port 32768 is now allowed for MariaDB.</p> Patrick Laimbockhttps://mariadb.com/kb/en/using-the-s3-storage-engine/+comments/4779
