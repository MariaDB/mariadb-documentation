MariaDB Knowledge Base Comments for: MariaDB 5.5 - Centos 7 - last version not availablehttps://mariadb.com/kb/en/mariadb-55-centos-7-last-version-not-available/+comments/feed/en-usMon, 09 Dec 2024 13:47:37 +0000Re: MariaDB 5.5 - Centos 7 - last version not availablehttps://mariadb.com/kb/en/mariadb-55-centos-7-last-version-not-available/+comments/3778<p>The MariaDB yum repositories are updated every time a release is made and we generally keep the most recent three versions of a given MariaDB branch in our repositories.</p> <p>CentOS, and by extension RedHat, update MariaDB in their repositories according to their own schedule, which we have no control over. Because of this, the version of <a href="/kb/en/what-is-mariadb-55/">MariaDB 5.5</a> in the CentOS/RHEL <code>base</code> repository may be a few versions out of date compared to what you will find in the MariaDB repositories.</p> <p>With a <a href="/kb/en/what-is-mariadb-55/">MariaDB 5.5</a> <code>/etc/yum.repos.d/mariadb.repo</code> file in place, you can see all of the <a href="/kb/en/what-is-mariadb-55/">MariaDB 5.5</a> versions currently available from both the MariaDB repositories and the CentOS base repository:</p> <pre class="fixed">[buildbot@centos74-amd64 ~]$ sudo yum list --showduplicates mariadb-server Available Packages MariaDB-server.x86_64 5.5.60-1.el7.centos mariadb MariaDB-server.x86_64 5.5.61-1.el7.centos mariadb MariaDB-server.x86_64 5.5.62-1.el7.centos mariadb mariadb-server.x86_64 1:5.5.60-1.el7_5 base </pre><p>So currently (Jan 2019) in the <a href="/kb/en/what-is-mariadb-55/">MariaDB 5.5</a> repository <a href="/kb/en/mariadb-5560-release-notes/">MariaDB 5.5.60</a>, 5.5.61, and 5.5.62 are available; and in the CentOS <code>base</code> repository only <a href="/kb/en/mariadb-5560-release-notes/">MariaDB 5.5.60</a> is available.</p> <p>As to why there is no <code>mariadb.service</code> file, that is because that is something CentOS/RedHat add to their mariadb-server 5.5.x packages. We only include a mariadb.service systemd file for CentOS 7 (located at <code>/usr/lib/systemd/system/mariadb.service</code>) for our <a href="/kb/en/what-is-mariadb-101/">MariaDB 10.1</a> and higher packages.</p> Daniel Bartholomewhttps://mariadb.com/kb/en/mariadb-55-centos-7-last-version-not-available/+comments/3778
