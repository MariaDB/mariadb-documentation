See the <a href="https://mariadb.org/about/#maintenance-policy">MariaDB Server maintenance policy</a> for details about each release and its EOL.</p> Ian Gilfillanhttps://mariadb.com/kb/en/maria-db-5-tot-10/+comments/6343
