Version number "10" breaks Delphi XE5/FireDAC

I just installed the latest version, switching from version 5. Sadly, because MariaDB now reports a Major version number of 10, it can no longer be used to Delphi/FireDAC.

[FireDAC][Phys][MySQL]-1101. Unsupported MySQL version [1000110000]. Supported are client and server from v 3.20 to v 6.2

Would there be any way to force MariaDB to report a Version in the '6' range? I assume there will be other software which will no longer work because of this single issue.

Bart Kindt SARTrack Limited New Zealand

Answer Answered by Ian Gilfillan in this comment.

I'm not aware of a way to force MariaDB to report a false version number outside of changing and compiling it yourself.

This is best done on the application side.
