MariaDB Knowledge Base Comments for: SEThttps://mariadb.com/kb/en/set/+comments/feed/en-usMon, 20 Jan 2025 03:10:21 +0000Re: SET evalution orderhttps://mariadb.com/kb/en/set/+comments/5670<p>The documentation should specify whether evaluation order can be relied on or not. Are SET statements always evaluated left-to-right? i.e.</p> <pre class="fixed">SET @a:=0, @b:=0; SET @a:=1, @b:=2, @c:=@a+@b; </pre><p>After execution, is @c guaranteed to be 3, or might it be 0, 1, or 2?</p> <p>(It's well-documented that in a SELECT, order of evalution is NOT well defined, so if the 2nd SET were a SELECT instead, all values from 0 to 3 might be possible. But this needs to be documented for SET also, as it's a different statement with possibly different semantics!)</p> Dean Trowerhttps://mariadb.com/kb/en/set/+comments/5670
