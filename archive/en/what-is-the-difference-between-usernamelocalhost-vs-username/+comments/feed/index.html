MariaDB Knowledge Base Comments for: What is the difference between '<username>'@localhost vs '<username>'@'%'?https://mariadb.com/kb/en/what-is-the-difference-between-usernamelocalhost-vs-username/+comments/feed/en-usMon, 09 Oct 2023 23:53:38 +0000Re: What is the difference between '<username>'@localhost vs '<username>'@'%'?https://mariadb.com/kb/en/what-is-the-difference-between-usernamelocalhost-vs-username/+comments/6291<p>The <code>@localhost</code> user can only connect to MariaDB from the same machine where the server is located. The <code>@%</code> user is allowed to connect from anywhere. If you were to use something like <code>'username'@'192.168.0.123'</code> then the user would be only allowed to connect from that IP address.</p> <p>What the user is allowed to do on the database is controlled with the <a href="/kb/en/grant/">GRANT</a> SQL command. By default newly created userd are allowed to do very little and access to the tables needs to be granted.</p> Markus Mäkelähttps://mariadb.com/kb/en/what-is-the-difference-between-usernamelocalhost-vs-username/+comments/6291
