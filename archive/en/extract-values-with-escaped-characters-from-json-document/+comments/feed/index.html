MariaDB Knowledge Base Comments for: Extract values with escaped characters from JSON Documenthttps://mariadb.com/kb/en/extract-values-with-escaped-characters-from-json-document/+comments/feed/en-usTue, 02 Jan 2024 00:36:39 +0000Re: Extract values with escaped characters from JSON Documenthttps://mariadb.com/kb/en/extract-values-with-escaped-characters-from-json-document/+comments/5799<p>You need to use two escape characters. A single escape would be applied by the SQL parser, but you want to pass the escaped string down. So:</p> <pre class="fixed">Set @json = '{&#34;Item_Name&#34;:&#34;60\\&#34; Table&#34;, &#34;Item_ID&#34;:&#34;1&#34;}'; select json_value(@json,'$.Item_Name') as Item_Name , json_value(@json,'$.Item_ID') as ID; +-----------+------+ | Item_Name | ID | +-----------+------+ | 60&#34; Table | 1 | +-----------+------+ </pre>Ian Gilfillanhttps://mariadb.com/kb/en/extract-values-with-escaped-characters-from-json-document/+comments/5799
