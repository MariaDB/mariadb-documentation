MariaDB Knowledge Base Comments for: Problem installing MariaDB mariadb-11.4.0-winx64 on windows server 2016https://mariadb.com/kb/en/problem-installing-mariadb-mariadb-11-4-0-winx64-on-windows-server-2016/+comments/feed/en-usMon, 14 Apr 2025 13:01:31 +0000Re: Problem installing MariaDB mariadb-11.4.0-winx64 on windows server 2016https://mariadb.com/kb/en/problem-installing-mariadb-mariadb-11-4-0-winx64-on-windows-server-2016/+comments/6593<p>I'm afraid you'd need to install new version of Windows from Microsoft.</p> <p>The icuuc.dll/icuin.dll are from Windows 10, since version 1703 <a href="https://learn.microsoft.com/en-us/windows/win32/intl/international-components-for-unicode--icu-">https://learn.microsoft.com/en-us/windows/win32/intl/international-components-for-unicode--icu-</a></p> <p>Windows Server 2016 is based on version 1607, and this version predates ICU support on Windows.</p> <p>So, MariaDB used something Windows Server 2016 does not have, and WS2016 appears unsupported. But this is in line with MariaDB support policy, since that specific version of Windows is out of mainstream support, for the last 2 years already, and MariaDB only supports "in mainstream policy, as per MS", at least for the new MariaDB versions.</p> <p>Yes, we probably need to make MSI more clever about it.</p> Vladislav Vaintroubhttps://mariadb.com/kb/en/problem-installing-mariadb-mariadb-11-4-0-winx64-on-windows-server-2016/+comments/6593
