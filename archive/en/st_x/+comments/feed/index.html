MariaDB Knowledge Base Comments for: ST_Xhttps://mariadb.com/kb/en/st_x/+comments/feed/en-usMon, 17 Mar 2025 04:59:52 +0000Re: ST_Xhttps://mariadb.com/kb/en/st_x/+comments/3082<p>ST_X(point) returns a scalar number, not a Geometry type. Do not try to cast it from a WKB to Text or it will return NULL without errors or warnings!</p> <p>The same is probably true for ST_Y(), ST_Z(), and ST_M().</p> <p>Example:</p> <pre class="fixed">MariaDB [db]&gt; select ST_Y(GeomFromText('POINT(-1 1504310400)')); +--------------------------------------------+ | 1504310400 | +--------------------------------------------+ MariaDB [db]&gt; select ST_AsText(ST_Y(GeomFromText('POINT(-1 1504310400)'))); +-------------------------------------------------------+ | NULL | +-------------------------------------------------------+ </pre> https://mariadb.com/kb/en/st_x/+comments/3082
