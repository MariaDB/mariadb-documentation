MariaDB Knowledge Base Comments for: AWS Key Management Encryption Pluginhttps://mariadb.com/kb/en/aws-key-management-encryption-plugin/+comments/feed/en-usMon, 19 Aug 2024 04:34:27 +0000Re: AWS Key Management Encryption Pluginhttps://mariadb.com/kb/en/aws-key-management-encryption-plugin/+comments/4029<p>The AWS Key Management Plugin is <a href="https://jira.mariadb.org/browse/MDEV-18752?jql=project%20%3D%20MDEV%20AND%20component%20%3D%20%22Plugin%20-%20AWS%20key%20management%22">no longer available from the APT repo</a>. As stated in this article, "the only way to currently obtain the plugin is to install it from source."</p> <p>For those of you that use Docker, I created a <a href="https://hub.docker.com/r/creemama/mariadb-aws_key_management-plugin-build">Docker image</a> that builds the AWS Key Management Plugin from source. The resulting aws_key_management.so is compatible with the <a href="https://hub.docker.com/_/mariadb">Docker official images for MariaDB</a>.</p> <p>To make aws_key_management.so, do the following:</p> <pre class="fixed">docker run --name aws_key_management_build creemama/mariadb-aws_key_management-plugin-build:latest docker cp aws_key_management_build:/usr/local/src/build-mariadb/plugin/aws_key_management/aws_key_management.so . </pre><p>Examine the <a href="https://github.com/creemama/docker/blob/master/mariadb-aws_key_management-plugin-build/10.3/Dockerfile">Dockerfile</a> for the steps used to build the plugin.</p> Chris Topherhttps://mariadb.com/kb/en/aws-key-management-encryption-plugin/+comments/4029
