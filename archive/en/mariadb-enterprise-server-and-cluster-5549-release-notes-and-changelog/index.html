MariaDB Enterprise Server 5.5.49 Release Notes

This version of MariaDB Enterprise Server is built off MariaDB 5.5.49.

  • Fixes for the following security vulnerabilities:
    • CVE-2016-0647
    • CVE-2016-0648
    • CVE-2016-0666
    • CVE-2016-0643

Debian 6 (squeeze) has been discontinued.

MariaDB Enterprise Cluster 5.5.49 Release Notes

This version of MariaDB Enterprise Cluster is built off MariaDB Galera Cluster 5.5.49 and is mainly an upgrade to benefit from many fixes and improvements in MariaDB Galera 5.5.49, including some backported fixes, as detailed in the changelog.
