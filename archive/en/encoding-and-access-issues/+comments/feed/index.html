MariaDB Knowledge Base Comments for: Encoding and access issueshttps://mariadb.com/kb/en/encoding-and-access-issues/+comments/feed/en-usMon, 14 Oct 2024 12:55:54 +0000Re: Encoding and access issueshttps://mariadb.com/kb/en/encoding-and-access-issues/+comments/1373<p>About your issues:</p> <p>The access issue is not related to the CONNECT engine. I had the same when trying to access a table with the mysql client. It was fixed by issuing:</p> <p>FLUSH PRIVILEGES;</p> <p>After the GRANT command.</p> <p>The UTF8 issue comes from the fact that when reading the remote table the strings are automatically converted. To fix this issue, just remove the CHARACTER SET UTF8 and DEFAULT CHARSET=utf8 COLLATE=utf8_bin from the declaration of the CONNECT table.</p> Olivier Bertrandhttps://mariadb.com/kb/en/encoding-and-access-issues/+comments/1373
