The “Counters.def” file will have the details regarding the counters that were added/created in MONyog. The MONyog folder can be found at the following path based on the corresponding OS,</p> <p>Windows: C:\ProgramData\Webyog\MONyog\Counters.def</p> <p>In Linux (For .rpm): /usr/local/MONyog/Counters.def</p> <p>In Linux (For .tar.gz) Goto the directory where MONyog package is exctracted.. i.e., &lt;MONyog exctracted folder&gt;/Counters.def</p> <p>You may also write to support@webyog.com if you have any queries.
