MariaDB Knowledge Base Comments for: ST_Yhttps://mariadb.com/kb/en/st_y/+comments/feed/en-usMon, 18 Nov 2024 03:33:40 +0000Re: ST_Yhttps://mariadb.com/kb/en/st_y/+comments/3083<p>ST_Y(point) returns a scalar number, not a Geometry type. Do not try to cast it from a WKB to Text or it will return NULL without errors or warnings!</p> <p>The same is probably true for ST_X(), ST_Z(), and ST_M().</p> <p>Example:</p> <pre class="fixed">MariaDB [db]&gt; select ST_Y(GeomFromText('POINT(-1 1504310400)')); 1504310400 MariaDB [db]&gt; select ST_AsText(ST_Y(GeomFromText('POINT(-1 1504310400)'))); NULL </pre> https://mariadb.com/kb/en/st_y/+comments/3083
