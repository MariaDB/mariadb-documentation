MariaDB Knowledge Base Comments for: JSON_OBJECThttps://mariadb.com/kb/en/json_object/+comments/feed/en-usMon, 31 Mar 2025 03:31:45 +0000Re: JSON_OBJECThttps://mariadb.com/kb/en/json_object/+comments/6328<p>WARNING! JSON_OBJECT will flatten a sub JSON object/array if the key's value has conditional paths that return a non-JSON type: eg</p> <pre class="fixed">SELECT JSON_DETAILED( JSON_OBJECT( 'key' , IF( TRUE , JSON_ARRAY(1,2) , JSON_COMPACT('XXX') ) ) ) </pre><p>will return </p> <pre class="fixed">{ &#34;key&#34;: [ 1, 2 ] } </pre><p>but </p> <pre class="fixed">SELECT JSON_DETAILED( JSON_OBJECT( 'key' , IF( TRUE , JSON_ARRAY(1,2) , 'XXX' ) ) ) </pre><p>will return</p> <pre class="fixed">{ &#34;key&#34;: &#34;[1, 2]&#34; } </pre><p>because the JSON array has been flattened due to the competing path returning a string ... even though it will NEVER use that path.</p> Todd MICHAELhttps://mariadb.com/kb/en/json_object/+comments/6328
