CREATE DATABASE

Sintaxe

CREATE [OR REPLACE] {DATABASE | SCHEMA} [IF NOT EXISTS] db_name
    [create_specification] ...

create_specification:
    [DEFAULT] CHARACTER SET [=] charset_name
  | [DEFAULT] COLLATE [=] collation_name

Descrição

CREATE DATABASE cria um banco de dados com o nome dado. Para usar esta instrução, você precisa do privilégio CREATE para o banco de dados. CREATE SCHEMA é um sinônimo para CREATE DATABASE.
