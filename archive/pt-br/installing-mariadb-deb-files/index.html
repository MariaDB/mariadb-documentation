Instalando os arquivos .deb do MariaDB

Contents

  1. Instalando o MariaDB usando APT
  2. Instalando os arquivos .deb manualmente

Instalando o MariaDB usando APT

Para Debian e Ubuntu, é altamente recomendável instalar desde os repositórios, usando apt-get, aptitude, ou outro gerenciador de pacotes.

Primeiramente importe a chave gpg que se usa para assinar os repositórios. Essa chave permite o apt verificar a integridade dos pacotes que ele baixa.

O id da nossa chave de assinatura é 0xcbcb082a1bb943db e a impressão digital completa da chave é:

1993 69E5 404B D5FC 7D2F E43B CBCB 082A 1BB9 43DB

O aplicativo apt-key é uma maneira fácil de importar esta chave para o apt:

sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 0xcbcb082a1bb943db

Ao importar a chave de assinatura, tanto no Ubuntu quanto no Debian. O comando acima permanece igual.

Exemplo:
localhost:~# apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 0xcbcb082a1bb943db
Executing: gpg --ignore-time-conflict --no-options --no-default-keyring --secret-keyring /tmp/tmp.ASyOPV87XC --trustdb-name /etc/apt/trustdb.gpg --keyring /etc/apt/trusted.gpg --primary-keyring /etc/apt/trusted.gpg --recv-keys --keyserver keyserver.ubuntu.com 0xcbcb082a1bb943db
gpg: requesting key 1BB943DB from hkp server keyserver.ubuntu.com
gpg: key 1BB943DB: public key "Daniel Bartholomew (Monty Program signing key) <dbart@askmonty.org>" imported
gpg: no ultimately trusted keys found
gpg: Total number processed: 1
gpg:               imported: 1

Uma vez que a chave é adicionada você está pronto para adicionar o repositório apropriado.

Para gerar facilmente as entradas apropriadas no sources.list (também conhecido como linhas APT), use o nosso gerador online de sources.list.

Uma vez obtidas as entradas do seu sources.list, adicione elas para o seu /etc/apt/sources.list local. (e.g. with "sudo gedit /etc/apt/sources.list " or to a separate file under sources.list.d/ (e.g. with "sudo gedit /etc/apt/sources.list.d/mariadb.list ".

Você também pode usar a ferramenta Software Sources para adicionar linhas APT. Esta ferramenta se encontra sob o menu "Editar" no Centro de Software do Ubuntu. No gerenciador de pacotes Synaptic esta ferramenta é chamada "Repositórios" e encontra-se sob o menu "Configurações". Após executar a ferramenta escolha a aba "Outro Software", clique no botão "Adicionar..." e cole a linha APT (uma vez por cada uma das duas linhas).

Tendo a chave e as linhas APT no lugar, você agora pode executar:

sudo apt-get update

...para baixar a informação apt precisa saber como instalar MariaDB.

MariaDB pode então ser instalada com seu gerenciador de pacotes preferido, por exemplo:

sudo apt-get install mariadb-server-5.2

Instalando os arquivos .deb manualmente

Mesmo não sendo recomendado, é possível baixar e instalar os pacotes manualmente (p.e. sem um gerenciador de pacotes como o apt-get). Navegue nos diretórios sob o diretório repo/ no mirror mais perto de você. Os arquivos .deb estão em debian/pool/ e ubuntu/pool/, respectivamente.

Aqui estão os comandos que usamos em um sistema Debian 5 amd64 para instalar o MariaDB 5.1.42 (para outras distribuições baseadas no Debian deveria ser similar):

sudo apt-get update
sudo apt-get install libdbi-perl libdbd-mysql-perl psmisc
sudo dpkg --install mysql-common_5.1.42-mariadb73_all.deb
sudo dpkg --install libmariadbclient16_5.1.42-mariadb73_amd64.deb libmysqlclient16_5.1.42-mariadb73_amd64.deb \
mariadb-client_5.1.42-mariadb73_all.deb mariadb-client-5.1_5.1.42-mariadb73_amd64.deb mariadb-server_5.1.42-mariadb73_all.deb \
mariadb-server-5.1_5.1.42-mariadb73_amd64.deb
