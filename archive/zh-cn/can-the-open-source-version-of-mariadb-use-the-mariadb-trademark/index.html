Can the open-source version of MariaDB use the MariaDB trademark?

Can the open-source version of MariaDB use the MariaDB trademark?

Answer Answered by Ian Gilfillan in this comment.

The question is a little unclear, so let me try answer as best I can from the MariaDB Foundation perspective. By "open-source version" I presume you just mean MariaDB Server (sometimes referred to as MariaDB Community Server by MariaDB Corporation). There's also MariaDB Enterprise Server offered by MariaDB Corporation, and this is also open-source.

"Use" can be ambiguous, but the MariaDB Foundation have exclusive rights to use "MariaDB Server" and the logo to further the MariaDB Foundation mission. For further details, see https://mariadb.org/about/trademark/.
