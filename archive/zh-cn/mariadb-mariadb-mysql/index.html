推荐的 MariaDB / MySQL 电子书

Contents

  1. 针对 MariaDB / MySQL 最终用户
  2. 针对想对 MariaDB 和 MySQL 编码的开发人员

这里是推荐的 MariaDB / MySQL 电子书的列表。

我们在这里为了方便提供 Amazon.com 的链接，但是你也许可以在本地书店中找到这些书。

如果你有你喜欢的 MySQL / MariaDB 电子书，也请在评论栏中留下相关内容供大家参考。

针对 MariaDB / MySQL 最终用户

  • MariaDB Crash Course by Ben Forta
    • 面向想要学习 SQL 和 MariaDB 基础知识的人群。
    • 第一本 MariaDB 书籍！
    • 现已发货。在Amazon.com或你喜欢的书店购买。
  • SQL-99 Complete, Really by Peter Gulutzan & Trudy Pelzer.
    • 关于 SQL 99 标准的所有你想知道的东西。优秀的参考书！
    • 在 Knowledgebase 免费阅读！
  • MySQL (4th Edition) by Paul DuBois
    • 如果你想学习使用 MySQL / MariaDB，这是“默认”要读的书。
  • MySQL Cookbook by Paul DuBois
    • 大量使用 MySQL 的示例。像 Paul 的所有书籍一样，它物超所值，即使是“枯燥”的主题也很愉快的阅读体验。
  • High Performance MySQL, Second Edition, 作者：Baron Schwartz、Peter Zaitsev、Vadim Tkachenko、Jeremy D. Zawodny、Arjen Lentz、Derek J. Balling 等。
    • “High Performance MySQL 是构建快速、可靠的 MySQL 系统的权威指南。本书由经验丰富的专家撰写，他们在构建大型系统方面拥有多年实际经验，详细介绍了 MySQL 性能的各个方面，并专注于强健性、安全性和数据完整性。深入学习高级技术，让您发挥 MySQL 的全部实力。”（来自 O'Reilly的书籍描述）
  • MySQL Admin Cookbook
    • 一个快速的步骤指南，面向 MySQL 用户和数据库管理员，处理 MySQL 配置和管理中的实际挑战。
  • MySQL 5.0 Certification Study Guide, 作者：Paul DuBois、Stefan Hinz、Carsten Pedersen
    • 这是官方指南，涵盖通过两个 MySQL 认证考试。它适用于版本 5.0 的服务器，因此尽管错过了 MySQL 5.1 及更高版本（包括 MariaDB 5.1 及更高版本）中提供的所有功能，但为最终用户提供了良好的基础理解。

针对想对 MariaDB 和 MySQL 编码的开发人员

  • Understanding MySQL Internals by Sasha Pachev，前 MySQL AB 开发人员。
    • 这是我们已知唯一一本描述 MariaDB / MySQL 内部的书。任何希望能够进行 MariaDB 开发都必须读的一本书！
    • 这本书并没有覆盖所有的内容，其中一些内容也过时了，但是还是非常好的一本书。
  • MySQL 5.1 Plugin Development 作者：Sergei Golubchik 和 Andrew Hutchings
    • 如果你想为 MariaDB 开发插件，这边书将会是你的必读教程。本书的作者是 Sergei，为 MySQL 和 MariaDB 设计了插件结构。
