DBeaver

DBeaver는 개발자와 SQL 프로그래머, 데이터베이스 관리자 및 분석가를 위한 무료 멀티 플랫폼 데이터베이스 도구입니다. MySQL, MariaDB, PostgreSQL, SQLite, Oracle, DB2, SQL Server, Sybase, Teradata, Netezza 등 모든 인기있는 관계형 데이터베이스를 지원합니다. 또한 Enterprise Edition에서는 MongoDB, Cassandra, Redis, Apache Hive와 같은 NoSQL 데이터베이스도 지원합니다.

기본 기능에 대한 간략한 목록은 아래와 같습니다:

  • 데이터 뷰어 및 편집기 : 정렬, 필터링, 이미지 표시, 선택한 데이터 내보내기 등.
  • 메타 데이터 브라우저 : 기존 테이블, 뷰, 컬럼, 인덱스, 프로 시저, 트리거, 스토리지 엔티티 (테이블 스페이스, 파티션 등), 보안 엔티티(사용자, 역할 등)를 보고 편집 할 수 있습니다.
  • 데이터 전송 : 다양한 형식 (CSV, HTML, XML, XLS, XLSX)으로 파일 내보내기 및 가져오기.
  • ER 다이어그램 : 데이터베이스/스키마 (다이어그램에는 모든 스키마 테이블이 포함됨) 또는 단일 테이블에 대한 ER 다이어그램을 자동으로 생성하고 적절한 파일 형식으로 내보낼 수 있습니다.
  • SQL 편집기 : 모든 스크립트를 폴더로 구성하거나, 특정 스크립트에 대한 데이터베이스 연결을 재할당 할 수 있습니다.
  • 데이터 및 메타 데이터 검색 : 선택한 모든 테이블/뷰에 대해 전체 텍스트 데이터 검색.
  • 데이터베이스 구조 비교 : 객체 구조를 비교할 수 있습니다.

DBeaver는 활발하게 개발되고 유지되고 있습니다. 이 프로젝트의 핵심 목표는 사용성이므로, 프로그램 UI는 신중하게 설계 및 구현되었습니다. 모든 사용자는 GitHub 페이지에서 버그 리포트와 기능 요청을 보낼 수 있습니다.
