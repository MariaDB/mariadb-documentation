About the MariaDB Foundation

MariaDB 재단은 오직 한 사람 또는 한 회사만이 MariaDB/MySQL 개발을 주도하지 않도록 보장합니다. MariaDB 재단은 MariaDB 코드의 관리자이자, MariaDB 커뮤니티의 보호자입니다.

MariaDB 재단은 MariaDB Server 메인 프로젝트의 소유자이며, mariadb.org를MariaDB 재단은 오직 한 사람 또는 한 회사만이 MariaDB/MySQL 개발을 주도하지 않도록 보장합니다. MariaDB 재단은 MariaDB 코드의 관리자이자, MariaDB 커뮤니티의 보호자입니다.

MariaDB 재단은 MariaDB Server 메인 프로젝트의 소유자이며, mariadb.org를 소유하고 있습니다. 이로서 공식 MariaDB 개발 트리가 MariaDB 개발자 커뮤니티에 열려 있도록 합니다.

목표

MariaDB가 커뮤니티에서 활발하게 개발되도록 하기 위해

  • MariaDB 도입을 증가시킵니다
  • 지속 가능한 빌드, 테스트와 배포를 위한 높은 수준의 노력을 보장합니다
  • 커뮤니티의 패치가 리뷰되고 적용되도록 보장합니다
  • 커뮤니티의 의견을 보증하고 개발 규칙을 정의합니다
  • MariaDB와 MySQL의 호환성을 유지합니다
  • mariadb.org를 유지합니다.

MariaDB를 제품에 사용하고 있고 MariaDB가 항상 활발하게 개발되기를 바라신다면, 스폰서 또는 MariaDB 재단 멤버가 되는 것을 권합니다!

MariaDB 재단은 MySQL 소스코드를 포함해 모든 커뮤니티 패치들이 MariaDB에 병합되도록 합니다. MariaDB의 빌드와 QA를 수행하고 방대한 MariaDB 문서를 제공하는 것 또한 MariaDB 재단의 몫입니다.

이 작업을 가능하게 해 주신 현재 멤버 및 스폰서 분들께 감사드립니다!
